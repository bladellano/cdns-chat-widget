🎨 Exemplos de Customização do Chat Widget

💼 Suporte Técnico

Ideal para equipes de suporte ao cliente com design profissional em azul.

Ver Exemplo 
window.nyxChatConfig = {
  primaryColor: '#2196f3',
  botName: 'Suporte Técnico',
  position: 'bottom-right'
};

🛒 Equipe de Vendas

Design vibrante em laranja para equipes comerciais e vendas.

Ver Exemplo 
window.nyxChatConfig = {
  primaryColor: '#ff5722',
  botName: 'Assistente de Vendas',
  position: 'bottom-left'
};

✨ Tema Personalizado

Roxo elegante com personalização completa de mensagens.

Ver Exemplo 
window.nyxChatConfig = {
  primaryColor: '#9c27b0',
  botName: 'Nina IA',
  buttonIcon: '✨',
  welcomeMessage: 'Oi! 👋'
};

📚 Como usar

Cada exemplo demonstra diferentes configurações do widget. Copie o código e ajuste conforme suas necessidades.

Opções disponíveis: primaryColor, botName, position, buttonIcon, welcomeMessage, placeholder, webhookId
